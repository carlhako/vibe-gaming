Request a new game

Describe the game you want. DeepSeek will generate it in the background — this can take up to about 20 minutes, and you'll be taken to a status page that updates automatically. That page's URL is unique to this request, so you can bookmark it and check back later if you close the tab.

Games are generated by DeepSeek, an AI model — not a full game engine, so don't expect full-blown 3D games. Aim for simple 2D games you can iterate on: Snake, a Breakout clone, a top-down maze collector, a one-screen platformer, an Asteroids-style shooter. Get something simple working first, then use Enhance to add depth.
